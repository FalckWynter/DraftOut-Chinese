# Draftout 简体中文汉化

这里是 Minecraft Mod ，Draftout 的简体中文任务文本汉化文件。

2026/7/1 更新
包含 v1.10.3 和 v1.11.1 两个版本的文件。
需要注意的是，v1.11.1大改了汉化翻译的key结构，所以两个版本的汉化文件不通用。

本仓库只包含翻译文件，不包含 Draftout MOD 本体、代码、贴图或其他原始资源。Draftout 归原作者所有。

## 使用方法

推荐将 `assets/draftout/lang/zh_cn.json` 放入资源包，或手动加入原 MOD JAR 的同一路径：

```text
assets/draftout/lang/zh_cn.json
```

进入游戏后，将语言切换为“简体中文”。

## 文件说明

- `zh_cn.json`：根目录副本，方便查看和下载。
- `assets/draftout/lang/zh_cn.json`：Minecraft/Fabric 资源路径版本。
